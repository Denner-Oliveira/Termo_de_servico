# Política de Privacidade

**Última atualização:** 03 de julho de 2026

## 1. Informações Coletadas

O Aplicativo poderá processar apenas as informações necessárias para seu funcionamento, tais como:

* IDs de usuários, servidores, canais e cargos do Discord.
* Configurações definidas pelos administradores do servidor.
* Dados necessários para executar comandos solicitados pelos usuários.

O Aplicativo **não coleta** mensagens privadas, senhas ou informações de pagamento.

## 2. Finalidade

Os dados são utilizados exclusivamente para:

* Executar os recursos do Aplicativo.
* Armazenar configurações quando necessário.
* Melhorar a estabilidade e o funcionamento do serviço.

## 3. Compartilhamento

Os dados não são vendidos, alugados ou compartilhados com terceiros, exceto quando exigido por lei.

## 4. Armazenamento

As informações são armazenadas apenas pelo tempo necessário para o funcionamento do Aplicativo.

## 5. Segurança

São adotadas medidas razoáveis para proteger os dados armazenados, porém nenhum sistema é completamente seguro.

## 6. Exclusão de Dados

Caso deseje solicitar a exclusão de dados relacionados ao Aplicativo, entre em contato pelo e-mail informado abaixo.

## 7. Alterações nesta Política

Esta Política poderá ser atualizada periodicamente. A versão mais recente estará sempre disponível nesta página.
